### What was used on this project

- HTML
- CSS
- JavaScript
- Jquery
- SoketIO
- Nodejs
- Express

- Clone the Repo

```
git clone git@github.com:Moises6669/Tic-Tac-Toe_Online.git
```

- Install the dependencies

```
cd tic-tac-toe-online
npm install
```

- Run the server

```
npm run dev
```
